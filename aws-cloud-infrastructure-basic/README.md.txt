 Project Overview
This project demonstrates the creation of a basic cloud infrastructure using AWS services.
It is designed for beginners and fresher-level Implementation Engineer roles.

 Services Used
- Amazon EC2
- Amazon VPC
- Security Groups
- IAM
- Amazon S3
- Application Load Balancer

 Architecture Description
- Created a VPC with public subnet
- Launched EC2 instance inside the subnet
- Configured Security Groups for SSH and HTTP
- Used IAM for secure access control
- Stored files in S3 bucket
- Configured Load Balancer to distribute traffic

 Security Features
- Key pair authentication for EC2
- Security group rules (stateful firewall)
- Private S3 bucket access
- IAM least privilege principle

 Outcome
Successfully designed and implemented a basic AWS cloud infrastructure suitable for hosting applications.
